# モダナイイベント準備の思考整理

## 1. イシュー定義
「モダナイイベントを成功させるために、準備の漏れをなくし、当日の運営を円滑にするにはどうすればよいか？」

## 2. イシューツリー展開
- **A. 物理的準備（印刷・備品）**
  - タイムキーピング資料、座席表、QRコード、サインペン、飲食
- **B. ステークホルダー管理（対応事項）**
  - 挨拶者確定、欠席者把握、飲食確認
- **C. コンテンツ・資料準備**
  - 挨拶資料、タイムテーブル、パネル、告知、アンケート・販売QR

## 3. Why So / So What 再帰分析

### 【要素A：物理的準備】
- **【第1段階】事実（What）**: タイムキーピングやQRコードなどの印刷物、サインペン等の備品が必要。
- **【第2段階】Why So? ①**: 当日の運営を秒単位で管理し、来場者の動線をスムーズにするため。
- **【第3段階】Why So? ②**: 運営スタッフ間の認識齟齬を防ぎ、トラブルを未然に防ぐ必要があるため。
- **【第4段階】Why So? ③**: イベントのクオリティが主催団体の信頼性に直結するため。
- **【第5段階】Root Why（最深因）**: 「細部へのこだわり」が参加者の満足度とブランド価値を決定づけるから。

### 【要素A：So What連鎖】
- **【第1段階】事実（What）**: 物理的準備の徹底。
- **【第2段階】So What? ①**: 当日の運営に余裕が生まれ、突発的な事態に対応できる。
- **【第3段階】So What? ②**: 登壇者や参加者がコンテンツに集中できる環境が整う。
- **【第4段階】So What? ③**: イベント後のアンケート結果や次回の集客にポジティブな影響を与える。
- **【第5段階】Final So What（最終示唆）**: 徹底した準備は、無形の信頼を形にするプロセスである。

---

### 【要素B：ステークホルダー管理】
- **【第1段階】事実（What）**: 挨拶者の確定や欠席者の把握が必要。
- **【第2段階】Why So? ①**: プログラムの進行を確定させ、当日の混乱を避けるため。
- **【第3段階】Why So? ②**: 重要なステークホルダーへの礼を失せず、関係性を維持するため。
- **【第4段階】Why So? ③**: イベントが単なる発表の場ではなく、コミュニティ形成の場であるため。
- **【第5段階】Root Why（最深因）**: イベントの本質は「人」であり、配慮の欠如は目的の喪失を意味するから。

### 【要素B：So What連鎖】
- **【第1段階】事実（What）**: ステークホルダー対応の完遂。
- **【第2段階】So What? ①**: 登壇者が安心して役割を果たせる。
- **【第3段階】So What? ②**: 欠席者へのフォローなど、イベント外でのコミュニケーションが活性化する。
- **【第4段階】So What? ③**: 主催団体のネットワークが強化され、次なるプロジェクトへの布石となる。
- **【第5段階】Final So What（最終示唆）**: 人への配慮が、イベントを単なる点から線（継続的な関係）に変える。

## 4. 構造メモ生成
- **対応関係**: 物理的な準備（A）はイベントの「土台」を支え、ステークホルダー管理（B）はイベントの「魂」を吹き込む。資料準備（C）はその両者をつなぐ「メディア」として機能する。
- **気づき（Insight）**: 印刷物やQRコードの準備は単なる作業ではなく、参加者の体験をデザインする重要なタッチポイントである。
- **仮説（Hypothesis）**: 最終挨拶者が未確定であることは、全体の締まりに影響する可能性があるため、早急な意思決定が必要。
- **次のアクション（Next Action）**: 
  1. 最終挨拶者の即時確定と連絡。
  2. 印刷物のチェックリスト作成と一括発注。
  3. 欠席者への当日配布資料の送付段取り。
