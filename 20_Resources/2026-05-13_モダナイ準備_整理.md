# モダナイ準備の思考整理と実行計画

## 1. イシュー定義
「モダナイイベントの成功に向け、物理的準備・ステークホルダー対応・資料作成の3軸をどのように同期させ、当日運営の質を最大化するか？」

## 2. イシューツリー展開
- **A. 物理的・機材準備（#実行 含む）**
  - タイムキーピング、座席表、AmazonQRのパワポ作成・印刷
  - サインペン、飲食等の備品確保
- **B. ステークホルダー管理**
  - 最終挨拶者の確定、欠席者（明安、北國、日新火災）の反映
- **C. コンテンツ・資料準備**
  - 各種挨拶資料、パネル、告知、アンケート・販売QRの作成

## 3. Why So / So What 再帰分析

### 【要素A：物理的・機材準備】
- **【第1段階】事実（What）**: パワポ作成が必要な印刷物（タイムキーピング等）と備品がある。
- **【第2段階】Why So? ①**: 運営の各フェーズで視覚的なガイドが必要だから。
- **【第3段階】Why So? ②**: スタッフや参加者の迷いをゼロにし、スムーズな進行を実現するため。
- **【第4段階】Why So? ③**: 準備不足は当日のパニックとブランド毀損に直結するため。
- **【第5段階】Root Why（最深因）**: 完璧な準備こそが、イベントという「一期一会」の場への敬意であるから。

### 【要素A：So What連鎖】
- **【第1段階】事実（What）**: 物理準備の完遂。
- **【第2段階】So What? ①**: 当日の運営が「自動運転」に近い状態で回り始める。
- **【第3段階】So What? ②**: 主催者が参加者との対話や、より高度なホスピタリティに時間を割ける。
- **【第4段階】So What? ③**: 参加者の「心地よさ」が最大化され、コミュニティの熱量が高まる。
- **【第5段階】Final So What（最終示唆）**: 優れた運営は、コンテンツの価値を数倍に引き上げる増幅器である。
